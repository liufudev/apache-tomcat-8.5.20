最美海岸线公益徒步活动激情开走

张小小2017-03-04 15：00

2013年5月，本报开展了“红灯记———拒绝不文明过马路”大型公益志愿活动。 晨报记者带头走上街头“劝阻行人闯红灯，提倡文明出行”，之后，来自政府机关、企事业单位、 志愿团体等社会各界力量，纷纷加入到这个公益活动中。
在上班高峰时段，在人流密集的交通路口，在车辆集中的路段， 对闯红灯等不文明过马路行为进行义务劝导。
2015年5月12日，在这项公益活动开展两周年的日子，本报组织千余名大学生志愿者组成了“红绿灯”警示标志， 成功创造了世界纪录协会“世界最大人体‘红绿灯’警示标志”的世界纪录。
4年时间里，本报编辑记者组成的志愿团队与社会各界志愿者们一直坚持至今，目前已有近40万人上街参与了这项公益活动。
